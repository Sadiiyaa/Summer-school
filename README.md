# IoT & Drone Projects
## IIT Jammu Summer School 2026

This repository contains the projects developed during the **Summer School 2026 - IoT, Robotics & Drones** conducted at **Indian Institute of Technology (IIT) Jammu** in collaboration with **I3C IIT Jammu** and **Techible**. The program focused on hands-on learning, real-world applications, and industry-oriented project development in emerging technologies. :contentReference[oaicite:0]{index=0}

# Projects Included



##  Dual ESP32 Radar System

A real-time radar visualization system using two ESP32 boards and ultrasonic sensors.

### Features

- 180° Servo Sweep
- Ultrasonic Distance Measurement
- ESP-NOW Communication
- Python Radar Viewer
- Target Tracking
- Live Object Detection
- Dual Radar Synchronization

### Technologies

- ESP32
- HC-SR04
- Servo Motor
- ESP-NOW
- Python
- Pygame
